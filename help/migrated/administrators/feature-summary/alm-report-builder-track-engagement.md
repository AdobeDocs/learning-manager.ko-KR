# Report Builder 내 사용자 그룹별 참여 추적

이 보고서는 교육 관리자와 L&amp;D 관리자가 월간 가장 활성화된 사용자 그룹과 참여 트렌드를 파악하는 데 유용합니다. Group by 및 집계 함수가 있는 **활성 필드 사용자 그룹** 및 **등록** 데이터 집합을 사용하여 사용자 그룹당 매월 하나의 요약 행을 생성합니다.

## 사용자 그룹별 참여 보고서 작성

1. **Report Builder**&#x200B;를 시작하고 **보고서 만들기**&#x200B;를 선택합니다.
2. 이름을 입력하십시오(예: _사용자 그룹 MoM_&#x200B;별 참여).
3. **열 선택** 패널에서 **활성 필드 사용자 그룹**&#x200B;을 확장하고 **사용자 그룹 이름** 옆의 **+**&#x200B;을(를) 선택합니다. 열이 **선택한 열** 패널에 나타납니다.
4. **등록**&#x200B;을 확장하고 **등록 날짜** 옆의 **+**&#x200B;을(를) 선택합니다.
5. **소요 시간** 옆의 **+**&#x200B;을(를) 선택합니다. **편집**(연필) 아이콘을 선택하고 별칭 _총 소요 시간_&#x200B;을 입력합니다.
6. **학습 개체**&#x200B;를 확장하고 **등록 사용자 수** 옆에 있는 **+**&#x200B;을(를) 선택합니다. 편집 아이콘을 선택하고 별칭 _총 등록 수_&#x200B;를 입력합니다.
7. **선택한 열** 패널 상단에서 **그룹화 기준: 선택**&#x200B;을 선택합니다.
8. **등록 - 등록 날짜**&#x200B;를 선택하고 기간을 **월**(으)로 설정하십시오. **활성 필드 사용자 그룹 - 사용자 그룹 이름**&#x200B;을 선택합니다. 둘 다 태그로 나타납니다. **등록 - 등록 날짜(월)** 및 **활성 필드 사용자 그룹 - 사용자 그룹 이름**.
9. **등록 - 소요 시간** 행에서 **집계 기준**&#x200B;을 선택하고 **합계**&#x200B;를 선택합니다.
10. **학습 개체 - 등록 사용자 수** 행에서 **집계 기준**&#x200B;을 선택하고 **개수**를 선택합니다.
   ![](assets/image038.jpg)
11. **필터 추가**&#x200B;를 선택합니다. **등록 - 등록 날짜**&#x200B;를 선택하고 **최근 3개월**&#x200B;과 같은 상대 범위를 선택하거나 특정 날짜 범위를 입력하십시오.
12. **+ 정렬 추가**&#x200B;를 선택합니다. **등록 - 등록 날짜** 오름차순으로 정렬한 다음 **총 소요 시간**&#x200B;에 보조 정렬을 추가합니다.
13. **보고서 저장**&#x200B;을 선택하고 **작업** > **다운로드**&#x200B;를 선택하여 보고서를 다운로드합니다.

보고서에는 매월 사용자 그룹당 하나의 행이 있으며, 해당 기간에 대한 총 소요 시간 및 총 등록을 표시합니다.
