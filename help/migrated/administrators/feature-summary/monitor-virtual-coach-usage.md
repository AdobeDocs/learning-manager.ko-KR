# 가상 코치 사용 모니터링

Adobe Learning Manager에서 Virtual Coach 사용 데이터를 보고, 월간 활성 사용자(MAU) 크레딧 사용량을 추적하고, 학습자 성과 보고서에 액세스합니다.

## 계정에 대한 Virtual Coach 활성화

Virtual Coach는 Adobe Learning Manager의 추가 기능으로 사용할 수 있습니다. 구매 후 프로비저닝은 계정 관리자에게 이메일로 전송되는 활성화 키를 생성합니다.

1. 관리자 권한으로 Adobe Learning Manager에 로그인합니다.
2. 왼쪽 탐색 창에서 **결제** 페이지로 이동합니다.
3. **가상 코치** 섹션에서 이메일로 받은 활성화 키를 입력합니다.
4. **적용**을 선택합니다. 가상 코치가 계정에 대해 활성화되었습니다.
   ![](assets/virtual-coach-037.png)

활성화되면 기능이 활성화되었음을 확인하는 앱 내 알림을 받게 됩니다. 작성자가 즉시 시작할 수 있도록 4개의 샘플 역할극 시나리오가 콘텐츠 라이브러리에 자동으로 추가됩니다.

>[!NOTE]
>
>활성화 키는 프로비저닝 중에 자동으로 생성되며 이메일을 통해 공유됩니다. 활성화 키가 없는 경우 Adobe Learning Manager 고객 성공 관리자(CSM)에게 문의하십시오.

## MAU 크레딧 잔액 보기

월간 활성 사용자(MAU) 크레딧은 매월 Virtual Coach를 사용하는 고유 학습자 수를 계산합니다.

1. **결제** 페이지로 이동합니다.
2. **가상 코치** 섹션에서 **사용 세부 정보 보기**&#x200B;를 선택합니다.
3. **기간 선택** 드롭다운을 사용하여 검토할 날짜 범위를 선택합니다.
   ![](assets/virtual-coach-038.png)

   **전체 사용량** 테이블에는 다음이 표시됩니다.

   a. **사용 가능:** 구매한 총 MAU 크레딧\
   b. **사용됨:**&#x200B;크레딧 소비량 누계\
   c. **남은 기간:** 남은 계약 기간 동안 사용할 수 있는 크레딧

   **월별 사용량** 테이블은 월별 고유 활성 학습자 수를 표시합니다.

4. 전체 사용 데이터를 내보내려면 **자세한 보고서 다운로드**&#x200B;를 선택하세요.

## MAU 크레딧 사용 방법

MAU 크레딧은 학습자가 역월에 Virtual Coach 세션을 완료할 때 소비됩니다. 같은 달에 같은 학습자의 추가 세션은 추가 크레딧을 소비하지 않습니다.

| 시나리오 | 소비된 MAU |
|----------|--------------|
| 한 학습자가 1월에 5개 세션을 완료합니다. | 1 |
| 동일한 학습자가 1월과 2월 모두에서 Virtual Coach를 사용함 | 2(월간 1개) |
| 1월에 각 100명의 학습자가 1개의 세션을 완료함 | 100 |

_MAU 크레딧은 각 학습자가 완료한 세션의 수와 관계없이 월별 고유 학습자당 계산됩니다._

계약 기간 종료 시 사용하지 않은 크레딧은 소멸되며 이월되지 않습니다.

### 예 1: 단일 학습자, 여러 세션

시나리오: Sarah가 1월에 5번의 가상 코치 세션을 마칩니다.

* 1월의 세션: 5
* 소비된 MAU: 1
* 이유: Sarah는 몇 번 실시하든 1월 동안 단일 고유 사용자로 계산됩니다.

### 예 2: 동일한 학습자, 여러 달

시나리오: Sarah는 1월과 2월 모두 가상 코치를 사용한다.

* 1월의 세션: 3
* 2월의 세션: 2
* 소비된 MAU: 2(1월의 경우 1 + 2월의 경우 1)
* 이유: 각 달력 월은 별도로 계산됩니다.

### 예 3: 동일한 달의 여러 학습자

시나리오: 100명의 영업 담당자가 1월에 가상 코치 세션을 하나씩 완료합니다.

* 총 세션: 100
* 소비된 MAU: 100
* 이유: 각 고유 학습자는 해당 월에 하나의 MAU로 계산됩니다.

### 예 4: 시간 경과에 따른 팀 연습

시나리오: 50명으로 구성된 팀은 1년 내내 Virtual Coach를 사용합니다.

| 월 | 활성 학습자 | 이번 달에 소비된 MAU | 누적 MAU |
|------|----------------|--------------------------|-----------------|
| 일년 | 0 | 0 | 0 |
| 이월 | 5(5명이 연습하지 않음) | 5 | 5 |
| 3월 | 0(전체 50개의 연습된 게인) | 0 | 45 |
| 4월 | 0 | 0 | 75 |

## 가상 코치 보고서 보기

**관리** 섹션 > **보고서** 섹션 > **AI 보고서** 페이지는 조직 전체의 모든 Virtual Coach 활동에 대한 사용 현황 및 성능 데이터를 제공합니다. **가상 코치** 제목 아래에서 두 개의 보고서를 사용할 수 있습니다.

모든 보고서는 CSV 형식으로 내보내집니다. 보고서 생성은 데이터 크기에 따라 몇 분 정도 소요될 수 있습니다.

### 학습자 사용 요약

모든 학습자의 월별 사용 데이터를 포함합니다. 이 보고서를 사용하여 매월 Virtual Coach를 사용하는 학습자 수를 추적하고, MAU 크레딧 소비를 모니터링하고, 시간 경과에 따른 참여 경향을 파악할 수 있습니다.

### 세션 세부 정보

지난 90일 동안의 모든 학습자에 대한 세션 수준 데이터를 포함합니다. 이 보고서를 사용하여 학습자 모집단의 개별 세션 점수, 주제 범위 및 스타일 메트릭을 검토하고 추가 교육이나 콘텐츠가 필요할 수 있는 스킬 격차를 식별할 수 있습니다.

## 보고서 액세스 및 다운로드

1. 관리자 권한으로 Adobe Learning Manager에 로그인합니다.
2. 왼쪽 탐색 창에서 **보고서**&#x200B;를 선택합니다.
3. **AI 보고서**&#x200B;를 선택합니다.
4. **가상 코치** 섹션에서 다운로드할 보고서, **학습자 사용 요약** 또는 **세션 세부 정보**를 선택합니다.
   ![](assets/virtual-coach-039.png)
5. 메시지가 표시되면 날짜 범위를 선택한 다음 **계속**&#x200B;을 선택합니다.
6. 보고서는 CSV 파일로 자동으로 다운로드됩니다.
