# Adobe Learning Manager의 적응형 과정

Adobe Learning Manager의 적응형 과정을 사용하면 각 학습자가 속한 사용자 그룹을 기반으로 각 학습자에게 표시되는 모듈과 필요한 모듈을 제어하여 여러 사용자에게 하나의 과정을 제공할 수 있습니다.

각 역할, 지역 또는 준수 프로필에 대해 별도의 강의를 작성하지 않고 하나의 적응형 강의에서 적합한 학습자에게 적합한 콘텐츠를 동적으로 표시합니다.

## 적응형 강의로 해결할 수 있는 문제

대규모의 다양한 직원을 교육하는 조직은 공통적으로 직면하고 있는 과제에 직면하고 있습니다. 데이터 개인정보 보호, 직장 윤리 및 안전은 다양한 역할, 위치 또는 규정 준수 의무를 가진 학습자에게 전달해야 합니다.

중복 생성: 작성자는 거의 동일한 여러 강의를 유지 관리하고, 보고는 단편화되며, 핵심 콘텐츠가 변경되면 모든 사본을 업데이트해야 합니다.

적응형 강의는 작성자가 사용자 그룹과 연관된 모듈 수준에서 가시성 및 완료 규칙을 구성할 수 있도록 하여 이 문제를 해결합니다. 한 코스는 모든 관객에게 동시에 서비스를 제공한다.

### 일반적인 시나리오

- 규정 준수 과정에는 모든 직원을 위한 핵심 모듈과 관할 지역 관련 부록 모듈이 있습니다. 각 학습자는 해당 위치에 적용되는 추가 항목만 볼 수 있습니다.
- 신입 사원 과정은 사원, 관리자 및 계약자에게 서로 다른 모듈을 표시합니다. 각 역할은 관련된 내용만 봅니다.
- 안전 교육 과정은 올해 중반에 새로운 필수 모듈을 추가합니다. 관리자는 새로 고침 완료를 트리거하여 이전에 완료한 모든 학습자가 새로운 모듈을 준수하도록 해야 합니다.

### 실세계 사례

조직은 전체 인력에게 필수 규정 준수 과정을 공개합니다. 강의에는 다음 7가지 모듈이 포함됩니다.

- 두 개의 모듈은 모든 직원에게 적용됩니다.
- 두 개의 모듈은 사용자 관리자에게만 적용됩니다.
- 두 모듈은 기술 역할을 담당하는 개별 컨텐츠 작가에만 적용됩니다.
- 하나의 모듈은 선임 이사 이상에게만 적용됩니다.

## 모듈 가시성 및 완료 작동 방식

적응형 강의의 각 콘텐츠 모듈에는 두 가지 설정이 있습니다.

**표시 대상:** 모듈을 볼 수 있는 사용자 그룹. 이러한 그룹의 학습자는 강의에서 모듈을 보고 액세스할 수 있지만 **필수**&#x200B;에 있지 않으면 완료할 수 없습니다.

강의를 완료하는 데 모듈이 필요한 사용자 그룹 **필수:**&#x200B;개. **필수** 아래에 나열된 모듈은 해당 그룹에 자동으로 표시됩니다. 두 설정에 동일한 그룹을 추가할 필요가 없습니다.

모듈은 특정 시점에 학습자에게 제공되는 세 가지 상태 중 하나입니다.

| 상태 | 결정 방법 | 완료 예정 금액 |
|---|---|---|
| 필수 | 학습자가 **필수** 아래에 나열된 사용자 그룹에 있습니다. | 예 - 완료해야 합니다. |
| 선택 사항 | 학습자가 **표시 대상** 아래의 그룹에 있지만 **필수 대상**&#x200B;은 아닙니다. | 아니요 - 표시 및 액세스 가능하지만 필수는 아닙니다 |
| 숨김 | 학습자가 어느 한 설정 아래의 그룹에 없습니다. | 학습자에게 전혀 표시되지 않음 |

## 적응 과정의 특성

적응형 강의의 정의적 특성은 강의가 등록 시에만 국한되지 않고 학습자의 프로필을 지속적으로 평가하는 것이다.

등록된 동안 학습자의 사용자 그룹이 변경되면 다음을 수행합니다.

- 모듈이 더 이상 새 사용자 그룹 아래에 표시되지 않으면 즉시 사라집니다.
- 새로 표시되는 모듈이 새 사용자 그룹에 필수인 경우 해당 모듈이 완료 요구 사항에 추가됩니다.
- 이전에 필수 모듈이었던 것이 더 이상 필수 모듈이 아니면 해당 모듈 완료 요구 사항에서 제거됩니다.
- 이전에 완료한 모듈은 완료된 상태로 유지됩니다. 프로필 변경이 이미 완료된 작업을 재설정하지 않습니다.

### 자동 등록 취소

사용자 그룹 변경으로 학습자의 필수 모듈이 모두 제거되면 학습자는 강의에서 자동으로 등록 취소됩니다.

### 자동 완성

사용자 그룹 변경이 진행 중인 학습자에서 미완료 필수 모듈을 모두 제거하면 해당 학습자에 대해 강의가 자동으로 완료됩니다.

프로필 변경으로 학습자가 완료하지 못한 새 필수 모듈이 발생할 경우 관리자는 새로 고침 완료를 트리거하여 기존 완료를 롤백하고 학습자가 새 모듈을 완료하도록 요구할 수 있습니다.

## 무엇에 적응하고 무엇을 유지할지

적응 규칙은 **콘텐츠 모듈**&#x200B;에만 적용됩니다. 다음은 사용자 그룹에 관계없이 등록된 모든 학습자에게 적용됩니다.

- **사전 작업 모듈:** 핵심 콘텐츠가 시작되기 전에 모든 학습자에게 표시됩니다.
- **테스트 모듈:** 모든 학습자가 사용할 수 있습니다. 테스트 모듈을 완료하면 콘텐츠 모듈 상태에 관계없이 과정이 완료됩니다.
- **사전 요구 사항:** 강의에 사전 요구 사항이 구성되어 있으면 사용자 그룹에 관계없이 모든 학습자가 등록하기 전에 해당 사전 요구 사항을 충족해야 합니다. 사전 요구 사항은 자동 적용되지 않으며 특정 사용자 그룹으로 범위를 지정할 수 없습니다.

작업 지원 및 강의에 첨부된 리소스 또한 적응력이 없습니다. 등록된 모든 학습자에게 표시됩니다.

스킬, 게임화 점수 및 배지는 학습자의 첫 번째 강의 완료에 따라 부여되며 프로필 변경으로 인한 재완료의 영향을 받지 않습니다.

>[!NOTE]
>
>적응형 과정이 외부로 공유되는 상위 LO의 일부인 경우 적응형 과정은 하위 계정에서 정규 과정으로 복사됩니다.


## 기능 가용성

적응 과정 기능은 2계층 계정 수준 플래그로 제어됩니다. 계정에 이 기능을 활성화하려면 Adobe 계정 팀에 문의하십시오.

계정 플래그가 활성화되면:

- 강의를 작성하거나 편집할 때 **완료 및 표시 규칙** 토글을 사용할 수 있습니다.
- 토글을 활성화하면 적응형 구성 패널이 활성화됩니다.

**주의:** 적응형 기능 플래그를 사용하도록 설정하면 **되돌릴 수 없음**&#x200B;입니다. 계정 수준에서 활성화되면 비활성화할 수 없습니다.

## 카탈로그 공유

적응형 강의는 계정 내 카탈로그에 추가할 수 있습니다. 카탈로그를 피어 계정 외부로 공유할 때 적응형 강의는 공유 콘텐츠에서 자동으로 제외됩니다.

>[!NOTE]
>
>적응 과정이 포함된 학습 경로나 인증이 외부와 공유되면 수신 계정은 카탈로그에서 학습 경로나 인증을 볼 수 있지만 해당 학습 경로 또는 인증 내의 적응 과정은 나타나지 않습니다. 학습 개체가 완전히 제외된 것은 아닙니다. 적응형 강의 구성 요소만 공유 버전에서 제거됩니다. 수신 계정의 작성자는 공유된 학습 개체에 소스 버전보다 더 적은 모듈이 있을 수 있다는 점을 인지해야 합니다.

## 지원되는 구성

| 구성 | 지원됩니까? |
|---|---|
| 일반 학습 경로에서의 적응 과정 | 예(아래 참고 참조) |
| 유연한 학습 경로의 적응 과정 | 예 |
| 적응형 학습 경로에서 적응형 과정 | 아니오 |
| 자격증의 적응형 과정 | 예(반복 인증에 권장되지 않음) |
| 다중 등록 | 아니오 |
| 인스턴스 전환 | 예 |
| 카탈로그 공유(계정 간) | 아니오 |
| 사전 작업 또는 테스트 모듈의 가시성 규칙 | 아니오 |
| 핵심 콘텐츠 모듈의 가시성 규칙 | 예 |

>[!NOTE]
>
>적응형 과정이 **순서가** 학습 경로에 포함된 경우 사용자 그룹이 모듈의 표시 규칙과 일치하지 않아 적응형 과정에 표시되는 모듈이 없는 학습자는 해당 과정을 완료할 수 없습니다. 순서가 지정된 학습 경로에서, 이 작업은 모든 후속 항목에 액세스할 수 없도록 차단합니다. 이를 방지하려면 학습 경로에 등록된 모든 학습자가 해당 경로의 모든 적응형 강의에서 하나 이상의 모듈에 대한 가시성이 있는 하나 이상의 사용자 그룹에 속해 있어야 합니다.

또한 상위(중첩된) 학습 경로에 적응형 과정을 포함하는 학습 경로를 포함하지 마십시오. 이 구성에서 학습자가 적응형 과정에 표시 또는 필수 모듈이 없는 경우 포함된 플레이어가 응답하지 않아 나머지 콘텐츠를 통한 탐색이 방지될 수 있습니다. 이 동작은 향후 릴리스에서 다루어질 예정입니다.

>[!NOTE]
>
>사용자 그룹 변경으로 보이는 모든 모듈이 제거되었기 때문에 **일반** 학습 경로 내의 적응 과정에서 학습자가 자동 등록 취소되면 상위 학습 경로는 등록된 상태로 유지됩니다. 학습 경로는 자동 등록 취소되지 않습니다. 학습자는 학습 경로 내의 적응 강의에 더 이상 액세스할 수 없어도 성적 증명서에 등록된 학습 경로를 확인할 수 있습니다. 사용 사례에서 적응형 과정이 등록될 때 상위 학습 경로도 등록 취소해야 하는 경우 적응형 과정을 포함하기 위해 일반 학습 경로 대신 **적응형 학습 경로**&#x200B;를 사용하는 것이 좋습니다.

## 계정에 적응형 강의 활성화

작성자가 사용자 그룹 멤버십을 기반으로 다른 학습자에게 다른 모듈을 표시하는 강의를 만들 수 있도록 적응 학습을 켭니다.

## 활성화하기 전에

- **영구:** 활성화되면 계정에 대한 적응형 학습을 끌 수 없습니다.
- **강의와 학습 경로 모두에 동시에 영향을 미칩니다.** 적응 강의를 활성화하는 동일한 플래그는 적응 학습 경로도 활성화합니다.
- **기존 강의는 변경되지 않습니다.** 새로 생성된 강의만 적응형 강의로 지정할 수 있습니다. 기존 정규 강의는 자동으로 변환되지 않습니다.
- **작성자는 옵션을 즉시 확인할 수 있습니다.** 저장하면 적응형 강의 유형이 작성 작업 과정에 표시됩니다.
- **2계층 프로비전:** 계정이 적응형 학습에 대해 프로비전된 경우 옵션이 활성화되고 잠겼습니다. UI에서 변경할 수 없습니다. 계정이 프로비저닝되지 않은 경우 설정이 전혀 표시되지 않습니다. 프로비저닝을 요청하려면 Adobe에게 문의하십시오.

## 적응형 강의 활성화

1. 관리자 권한으로 Adobe Learning Manager에 로그인합니다.
2. 왼쪽 탐색 창에서 **설정**&#x200B;을 선택합니다.
3. **일반**&#x200B;을 선택합니다.
4. **표시 및 완료 규칙** 섹션으로 이동합니다. 조직에 대해 적응형 학습이 활성화된 경우 다음과 같이 옵션이 잠긴 것으로 표시됩니다.

![](assets/image_0001.png)

이제 적응형 학습이 계정에 대해 활성화됩니다. 작성자는 적응형 강의 및 적응형 학습 경로를 즉시 만들 수 있습니다.

## 활성화 후 변경되는 내용

적응형 학습을 활성화한 후:

- 작성자는 강의를 만들 때 기존의 정규 강의 유형과 함께 **콘텐츠 표시 및 완료 규칙** 옵션을 볼 수 있습니다.
- 적응형 강의의 각 콘텐츠 모듈은 사용자 그룹에 대한 **선택 사항** 및 **필수** 규칙으로 구성할 수 있습니다.
- 적응형 강의에 등록된 학습자는 사용자 그룹이 표시하는 모듈만 볼 수 있습니다.
- 기존의 모든 정규 강의는 그대로 유지됩니다.

## 문제 해결

- **표시 여부 및 완료 규칙 섹션이 설정**&#x200B;에 표시되지 않습니다. 이 기능은 백엔드에서 프로비저닝되어야 토글이 나타납니다. 액세스 권한을 요청하려면 Adobe 계정 담당자 또는 Adobe 지원 센터에 문의하십시오.
- **토글이 이미 사용하도록 설정되어 있고 잠긴 것으로 나타납니다.** 계정이 프로비전되었을 때 적응형 학습이 사용하도록 설정되었습니다. 조치가 필요하지 않습니다. 작성자는 이미 적응형 강의를 만들 수 있습니다.
