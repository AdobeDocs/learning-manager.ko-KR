# Report Builder에서 보고서 복제 및 재사용

보고서를 복제하면 모든 열, 별칭, 설정별 그룹화, 집계, 필터 및 정렬이 편집 가능한 새 보고서로 복사됩니다. 다른 카탈로그, 사용자 그룹 또는 기간에 대한 보고서 버전을 작성하려는 경우 이 옵션을 사용하십시오.

## 보고서 복제

1. 복제할 보고서를 엽니다.
2. **작업** > **복제**&#x200B;를 선택합니다. 보고서 사본이 편집 모드에서 &quot;\[원본 보고서 이름\]-Copy&quot; 이름으로 열립니다. 예를 들어, 관리자 전체의 구조화된 성능 보기 - 복사
3. 보고서의 새 이름을 입력합니다.
4. 필터를 업데이트하거나 열을 조정하거나 정렬을 변경하는 등의 변경을 수행합니다.
5. **보고서 저장**&#x200B;을 선택합니다.

복제된 보고서는 [보고서] 탭에 저장되며 원본과 독립적입니다.

## 템플릿 복제

**템플릿** 탭에서 템플릿을 복제할 수도 있습니다. 템플릿을 복제하면 **보고서** 탭에서 편집 가능한 새 보고서가 만들어집니다. 원본 템플릿은 변경되지 않은 상태로 유지됩니다.

1. **템플릿** 탭을 선택합니다.
2. 복사할 템플릿을 찾습니다.
3. **복제**&#x200B;를 선택합니다.

   ![](assets/report-builder-0051.png)

4. 이름을 입력하고 조정한 다음 **보고서 저장**&#x200B;을 선택합니다.

## 모범 사례

* 다양한 팀, 카탈로그 또는 기간에 대한 변형이 필요한 경우 테스트를 거친 보고서를 기본으로 복제합니다. 이는 처음부터 제작하는 것보다 빠르며 일관된 구조를 보장합니다.
* 원본과 무엇이 다른지 반영하도록 사본 이름을 즉시 변경합니다(예: &#39;규정 준수 보고서 사본&#39;이 아닌 &#39;VP-APAC에 의한 규정 준수&#39;).
* 템플릿을 복제한 후 다른 변경 작업을 수행하기 전에 이름을 업데이트하여 **보고서** 탭에서 보고서를 식별할 수 있도록 합니다.
