# 피드백 보고서

## 개요

Adobe Learning Manager의 피드백 보고서는 학습자가 학습 개체를 완료한 후 레벨 1(학습자 피드백)과 레벨 3(관리자 피드백)을 모두 수집합니다. 이 보고서는 학습자와 관리자의 주관적, 객관적 반응에 대한 구조화된 개요를 제시한다.

이 보고서는 이름, 이메일, 피드백 양식 이름, 버전, 언어 등의 학습자 세부 정보를 추적하고 관리자가 정의한 질문에 대한 응답을 캡처합니다. 또한 각 질문에 대한 학습자의 선택도 기록합니다(예: 강력하게 동의, 동의 또는 동의 안 함).

## 목적 및 이점

* 강의 품질과 전반적인 학습 효과를 향상하는 실행 가능한 통찰력을 제공합니다.
* 직접 사용자 피드백을 기반으로 탐색, 타이밍 및 전달 방법을 세분화합니다.

## 사용 사례

* **콘텐츠 문제를 빠르게 식별**: 관리자는 낮은 등급이나 반복된 부정적인 주석을 발견하고 지원 티켓 또는 에스컬레이션을 기다리지 않고 학습 개체를 업데이트할 수 있습니다.
* **교육 효과 측정**: 팀은 여러 과정 또는 버전에서 학습자 피드백을 비교하여 어떤 학습 개체가 잘 수행되고 있고 재작업해야 하는지 이해할 수 있습니다.
* **피드백 양식으로 학습자 참여 추적**: 관리자는 질문에 대답하거나 건너뛰는 학습자 수를 확인할 수 있으며, 이를 통해 응답 품질과 완료율을 향상시킬 수 있습니다.

## 피드백 보고서를 다운로드하는 방법

1. 관리자 권한으로 Adobe Learning Manager에 로그인합니다.
2. 왼쪽 탐색 메뉴에서 **[!UICONTROL 보고서]**를 선택합니다.
   ![](assets/select-report.png)
   _관리자 홈페이지에는 보고서 다운로드를 위해 강조 표시된 보고서 옵션이 표시됩니다_

3. 보고서에서 **[!UICONTROL 사용자 지정 보고서]**&#x200B;를 선택한 다음 **[!UICONTROL Excel 보고서]**&#x200B;를 선택합니다.
4. **[!UICONTROL 피드백 보고서]**를 선택합니다.
   ![](assets/select-feedback-report.png)
   _사용자 지정 보고서 섹션은 피드백 보고서를 선택하여 학습자 및 관리자 피드백 데이터에 액세스하는 옵션을 표시합니다_

5. **[!UICONTROL 모든 교육]** 또는 **[!UICONTROL 선택한 교육]** 및 날짜 범위를 선택합니다. 두 번째 옵션을 선택하면 최대 10개의 강의 또는 학습 경로를 추가하고 피드백 보고서를 생성할 수 있습니다. 또한 최대 1년 동안 보고서를 생성할 수 있습니다.
   ![](assets/feedback-report.png)
   _교육 범위를 선택하고 날짜 범위를 설정한 다음 다운로드하기 전에 번역 옵션을 선택하여 피드백 보고서를 구성합니다_

6. L1 피드백을 번역할 언어를 선택합니다. 객관적 질문과 그 응답은 해당 언어 버전이 명시적으로 정의될 때 선택된 언어로 번역된다. 선택된 언어로 명시적으로 정의된 주관적인 질문만 보고서에 나타납니다.  주관적인 질문에 대한 응답은 원래의 응답 언어로 되어 있을 것이다.
7. 보고서를 다운로드하려면 **[!UICONTROL 다운로드]**&#x200B;를 선택하십시오.

## 피드백 보고서에는 무엇이 포함되어 있습니까?

다음은 계정 수준 보고서의 기본 열입니다.

| 열 | 설명 |
|----|----|
| 피드백 유형 | 피드백이 학습자(L1)인지 관리자(L3)인지 나타냅니다. |
| 사용자 이름 | 교육을 완료한 학습자의 이름 |
| 사용자 전자 메일 | 학습자의 이메일 주소 |
| 교육 ID | 각 학습 객체(강의, 인증 또는 학습 경로)에 할당된 시스템에서 생성된 고유 식별자입니다 |
| 교육 이름 | 피드백이 제출되는 학습 항목의 이름 |
| 교육 인스턴스 | 교육의 인스턴스 이름(다중 인스턴스 강의의 경우) |
| 교육 유형 | 교육 유형(과정, 인증, 학습 경로) |
| 피드백 양식 유형 | 사용된 피드백 양식의 유형/범주(예: 혼합 강의, 자가 진행식 강의 및 강의실 강의) |
| 피드백 양식 이름 | 피드백 양식 이름 |
| 피드백 버전 | 피드백 양식의 버전 번호 |
| 현재 관리자 | 학습자 관리자의 이름 |
| 현재 관리자 전자 메일 | 학습자 관리자의 이메일 주소 |
| 완료 날짜 | 학습자가 교육을 완료한 날짜 |
| 피드백 날짜 | 학습자가 피드백을 제출한 날짜 |
| L1 피드백 원본 언어 | 학습자가 처음 L1 피드백을 제출한 언어 |
| L3 Likert 척도 질문 1 | 평가 척도를 사용하여 학습자의 교육 후 성과를 측정합니다 |
| L3 Likert 척도 응답 1 | 이 리커트 척도 질문에 대한 관리자의 응답 |
| L3 자유 텍스트 질문 1 | 관리자용 L3 피드백 양식에 자유 텍스트 질문이 추가됨. 선택 사항 또는 필수 항목으로 구성할 수 있습니다. |
| L3 자유 텍스트 응답 1 | 해당 자유 텍스트 질문에 대한 관리자의 응답 |

피드백 양식에 추가된 네 가지 유형의 질문에 따라 계정 수준 보고서에는 다음 열이 표시됩니다.

| 열 | 설명 |
|---|---|
| 강의 효과 | 양식에 미리 정의된 강의 효과 질문 |
| 강의 효과 대답 | 강의 효과 질문에 대한 학습자의 응답 |
| NPS 질문 1 | 첫 번째 순 프로모터 점수 질문 |
| NPS 범위 1 | 사용된 등급 척도(예: 0~10) |
| NPS 응답 1 | NPS 질문 1에 대한 학습자 평가 |
| Likert 질문 1 | 첫 번째 리커트 척도 질문 |
| Likert 대답 1 | Likert 질문 1에 대한 응답 |
| 텍스트 질문 1 | 양식의 첫 번째 개방형/텍스트 질문 |
| 텍스트 대답 1 | 텍스트 질문 1에 대한 학습자의 응답 |


>[!NOTE]
>
>계정 수준 보고서에는 학습자용으로 구성된 활성 필드도 포함됩니다.

학습 객체 레벨 보고서에는 다음 열이 표시됩니다.

| 열 | 설명 |
|---|---|
| 학습자 | 학습자의 이름 |
| 이메일 | 학습자의 이메일 주소 |
| 피드백 양식 이름 | 피드백 양식 이름 |
| 피드백 버전 | 피드백 양식의 버전 번호 |
| 학습자 언어 | 학습자가 선택한 언어 |

>[!NOTE]
>
>학습 객체 레벨 보고서에는 피드백 양식에 추가된 질문도 포함됩니다. 각 질문은 열 머리글로 표시되며 해당 질문에 대한 학습자의 응답은 아래의 해당 행에 표시됩니다.
