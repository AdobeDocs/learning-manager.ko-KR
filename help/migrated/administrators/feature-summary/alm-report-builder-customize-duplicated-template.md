# 복제된 Report Builder 템플릿 사용자 정의

템플릿을 복제한 후에는 저장 및 다운로드 전에 열을 추가 또는 제거하고, 필터를 업데이트하고, 열 이름을 변경하고, 정렬을 조정할 수 있습니다.\. 이 문서에서는 각 사용자 정의 단계에 대해 설명합니다.

시작하기 전에 **템플릿** 탭에서 템플릿을 복제하세요.

## 열 추가 및 제거

1. 편집 뷰에서 추가할 열이 포함된 데이터 세트를 찾습니다.
2. 열 이름 옆에 있는 **+**&#x200B;을(를) 선택합니다. 보고서 캔버스에 해당 열이 표시됩니다.
3. 예를 들어 같은 열을 두 번 이상 추가하려면 두 개의 다른 상태 값을 계산합니다. 해당 열에 대해 **+**&#x200B;을(를) 다시 선택합니다.
4. 열을 제거하려면 캔버스에서 열을 선택한 다음 제거 아이콘을 선택합니다.

## 열 순서 바꾸기 및 이름 바꾸기

1. 캔버스에서 원하는 위치로 열을 드래그합니다. 캔버스의 순서는 다운로드한 파일의 열 순서와 일치합니다.
2. 열의 이름을 바꾸려면 해당 별칭 필드를 선택하고 보고서에 열 머리글로 표시할 이름을 입력합니다.

## 필터 업데이트

1. 기존 필터 값을 변경하려면 필터 행을 선택하고 값을 편집한 다음 확인합니다.
2. 새 필터를 추가하려면 **필터 추가**&#x200B;를 선택하십시오.
3. 필터링할 필드를 선택합니다.
4. 연산자를 선택합니다. 사용 가능한 연산자는 필드의 데이터 유형에 따라 다릅니다.
5. 값 입력 또는 선택:
   * 문자열 필드의 경우 검색할 문자열을 입력하고 제안 중에서 선택합니다.
   * 목록 필드의 경우 선택기에서 하나 이상의 값을 선택합니다.
   * 날짜 필드에 날짜를 입력하거나 최근 90일과 같은 상대 범위를 선택합니다.
6. 필터를 제거하려면 해당 필터 행에서 삭제 아이콘을 선택합니다.

## 정렬 업데이트

1. 정렬할 열에서 정렬 아이콘 을 선택합니다.
2. **오름차순** 또는 **내림차순**&#x200B;을 선택합니다.
3. 추가 열을 기준으로 정렬하려면 각 열에 대해 반복합니다. 첫 번째 정렬된 열은 기본 열이며 추가 정렬은 관계 내에 적용됩니다.

>[!TIP]
>
>항상 한 가지 이상의 정렬을 적용합니다. 정렬하지 않으면 동일한 보고서의 다운로드 간에 행 순서가 다를 수 있습니다.

## 저장 및 다운로드

1. 저장 을 선택하여 변경 내용을 **보고서** 탭에 저장합니다.
2. **다운로드**&#x200B;를 선택하여 보고서를 생성합니다. 파일이 준비되면 인앱 알림을 받게 됩니다.
