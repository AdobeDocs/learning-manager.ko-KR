# Adobe Learning Manager에서 원클릭 등록 설정

## 원클릭 등록 작동 방식

원클릭 등록은 두 개의 Adobe Learning Manager(ALM) 기능이 함께 작동하므로
딥 링크는 ALM 내의 특정 강의 또는 학습 객체에 대한 다이렉트 URL을 제공합니다. 전자 메일, 인트라넷 포털 또는 서드파티 애플리케이션에 이러한 링크(모듈로 연결되는 직접 URL)를 포함할 수 있습니다. 링크를 클릭할 때 학습자가 아직 로그인하지 않은 경우 ALM에서 인증하라는 메시지를 표시한 다음 강의로 직접 이동합니다.

학습자가 작업 중이고 Slack 또는 팀을 사용 중이거나 여행, 회의 또는 영업 전화 등에 참석하기 전에 2분 내외의 간단한 재교육 과정이 필요할 때 일반적으로 유용합니다. 즉시 콘텐츠에 액세스하고 교육을 받을 수 있습니다.
등록 관련 서비스는 딥 링크가 과정 플레이어를 시작하기 전에 학습자를 과정에 자동으로 등록합니다. 이렇게 하면 학습자의 환경이 중단되는 수동 등록 단계가 제거됩니다.

학습자가 원클릭 등록 링크를 선택하면 ALM은 백그라운드에서 API를 통해 등록 후 딥 링크를 사용하여 강의로 리디렉션합니다. 강의 플레이어가 즉시 열립니다.

>[!NOTE]
>
>등록은 강의 레벨에서만 발생하며 상위 학습 객체(학습 경로 또는 인증)에서는 발생하지 않습니다.

## 모듈에 대한 딥 링크 생성

1. 관리자 권한으로 Adobe Learning Manager에 로그인합니다.
2. 왼쪽 탐색 창에서 **과정**을 선택합니다.
   ![](assets/one-click-enroll1.png)
3. 강의 선택
4. **인스턴스**를 선택합니다.
   ![](assets/one-click-enroll2.png)
5. 모듈의 딥 링크를 복사할 인스턴스에서 **모듈** 섹션을 선택합니다. 모듈 세부 정보는 인스턴스 하단의 확장 섹션에 표시됩니다.
   ![](assets/one-click-enroll3.png)
6. 링크를 복사할 모듈로 이동합니다.
   ![](assets/one-click-enroll4.png)
7. **링크 복사**&#x200B;를 선택합니다. 이제 딥 링크가 복사되었습니다. 이 딥 링크는 강의의 특정 모듈을 여는 링크입니다.

이제 이 딥 링크를 사용하여 학습자에게 보낼 수 있습니다.
