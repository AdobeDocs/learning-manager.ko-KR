# Adobe Learning Manager 데스크탑 앱 문제 해결

이 문서에는 Adobe Learning Manager 데스크탑 응용 프로그램을 설치하고 사용하는 동안 발생하는 일반적인 문제를 해결하는 기본적인 문제 해결 팁이 있습니다.

## 다음 작업을 수행할 수 없습니다. {#iamunabletodothefollowing}

+++Adobe Learning Manager 데스크탑 응용 프로그램을 다운로드할 수 없습니다.

1. 인터넷 연결 및 방화벽 설정을 확인하십시오.
1. [소셜 학습]에서 **[!UICONTROL 새 게시물]** 게시물을 작성합니다. 게시판이 없다면 먼저 게시판을 만드십시오.
1. 스크린샷, 오디오 레코딩, 비디오 레코딩, Learning Manager 갤러리와 같은 콘텐츠를 만들 수 있도록 표시되는 다음 게시물 단추 옵션 중 하나를 클릭합니다. 데스크탑용 Adobe Learning Manager 데스크탑 응용 프로그램을 다운로드할 수 있는 Adobe Learning Manager 데스크탑 응용 프로그램 페이지가 표시됩니다.
1. 책임자에 의해 [소셜 학습]이 활성화된 유효한 Adobe Learning Manager 계정이 필요합니다. 책임자가 웹 브라우저를 통해 다운로드를 비활성화했을 수도 있습니다. Adobe Learning Manager 데스크탑 앱 다운로드에 대한 자세한 내용은 Adobe Learning Manager 책임자에게 문의하십시오.

+++

+++Adobe Learning Manager 데스크탑 응용 프로그램을 설치할 수 없습니다.

1. 시스템이 최소 시스템 요구 사항을 충족하는지 확인합니다. 자세한 내용은 [데스크탑용 Adobe Learning Manager 앱 시스템 요구 사항](../learners/adobe-learning-manager-app-for-desktop/adobe-learning-manager-desktop-app-system-requirements.md).
1. Adobe Learning Manager 데스크탑 응용 프로그램의 이전 설치를 모두 정리합니다. 자세한 내용은 다음을 참조하세요.  [이전 설치를 정리하는 방법](#howtocleanuppreviousinstallationsofadobelearningmanagerdesktopapp) 를 참조하십시오.
1. 설치 프로세스 중 오류가 발생한 경우 다음을 참조하십시오. [응용 프로그램 로그를 찾는 방법](#howtofindapplicationlogs). 도움이 필요하면 Adobe Learning Manager 데스크탑 응용 프로그램 책임자에게 문의하십시오.

+++

+++Adobe Learning Manager 데스크탑 응용 프로그램을 시작할 수 없습니다.

1. Adobe Learning Manager 데스크탑 응용 프로그램이 다운로드 및 설치되어 있는지 확인합니다.
1. [소셜 학습]에서 **[!UICONTROL 새 게시물]** (게시판이 없으면 게시판을 만드십시오). 게시물 단추 옵션(스크린샷 생성, 오디오 레코딩, 비디오 레코딩, Adobe Learning Manager 갤러리) 중 하나를 클릭합니다. Learning Manager 데스크탑 응용 프로그램을 실행할 수 있는 Adobe이 표시됩니다.
1. 앱이 실행되지 않는 경우 Windows의 시작 메뉴 또는 Mac OS X의 Launchpad에서도 실행할 수 있습니다.

+++

+++Adobe Learning Manager 데스크탑 응용 프로그램에서 내 계정에 로그인할 수 없습니다.

1. 인터넷에 연결되어 있고 방화벽 설정이 Adobe Learning Manager 데스크탑 응용 프로그램을 차단하지 않는지 확인합니다.
1. [소셜 학습]이 활성화된 유효한 Adobe Learning Manager 학습자 계정이 있는지 확인합니다.
1. 여전히 로그인할 수 없는 경우 Adobe Learning Manager 데스크탑 응용 프로그램을 종료했다가 다시 실행한 다음 다시 시도해 봅니다.
1. 도움이 필요하면 Adobe Learning Manager 책임자에게 문의하십시오.

+++

+++Adobe Learning Manager 데스크탑 응용 프로그램에 나열된 내 웹캠/마이크가 표시되지 않습니다.

1. 웹캠/마이크가 시스템에 제대로 연결되어 있고 제대로 작동하는지 확인합니다.
1. 웹캠/마이크용 최신 드라이버가 설치되어 있는지 확인합니다. 일부 장치의 경우 전용 드라이버가 없으면 제대로 작동하지 않습니다.
1. 응용 프로그램 환경 설정을 재설정한 다음, Adobe Learning Manager 데스크탑 응용 프로그램을 다시 실행하고 다시 시도합니다. 자세한 내용은 다음을 참조하세요. [응용 프로그램 환경 설정 재설정 방법](#howtoresetapplicationpreferences).
1. Mac OS X Mojave 10.14에 있는 경우 Adobe Learning Manager 데스크탑 응용 프로그램에 권한을 부여하여 웹캠/마이크에 액세스합니다. 자세한 내용은 다음을 참조하세요. [OSX Mojave에서 웹캠/마이크 권한을 설정하는 방법](#howtosetwebcammicrophonepermissionsonMacOSXMojave).

+++

+++Adobe Learning Manager 데스크탑 응용 프로그램에서 게시물을 게시할 수 없습니다.

1. Adobe Learning Manager 관리자에 의해 [소셜 학습]이 활성화된 유효한 Adobe Learning Manager 학습자 계정이 있는지 확인합니다.
1. 응용 프로그램 환경 설정을 재설정한 다음, Adobe Learning Manager 데스크탑 응용 프로그램을 다시 실행하고 다시 시도합니다. 자세한 내용은 다음을 참조하세요. [응용 프로그램 환경 설정 재설정 방법](#howtoresetapplicationpreferences).
1. 게시 중 오류가 발생하면 고급 로깅을 사용하십시오. 자세한 내용은 다음을 참조하세요. [고급 로깅을 사용하는 방법](#howtoenableadvancedlogging), Adobe Learning Manager 데스크탑 응용 프로그램을 다시 시작하고 오류를 초래하는 위의 단계를 다시 실행하십시오. 도움이 필요하면 최신 응용 프로그램 로그를 Adobe Learning Manager 관리자에게 보내십시오. 자세한 내용은 다음을 참조하세요. [응용 프로그램 로그를 찾는 방법](#howtofindapplicationlogs).

+++

+++이전 프로젝트를 보거나 열 수 없습니다.

1. Adobe Learning Manager 계정으로 생성된 프로젝트는 프로젝트가 생성된 것과 동일한 컴퓨터에서만 볼 수 있습니다.
1. 응용 프로그램 환경 설정을 재설정한 다음, Adobe Learning Manager 데스크탑 응용 프로그램을 다시 실행하고 다시 시도합니다. 도움이 필요한 경우 다음을 참조하십시오. [응용 프로그램 환경 설정 재설정 방법](#howtoresetapplicationpreferences).
1. 프로젝트를 여는 동안 오류가 발생하면 고급 로깅을 사용하십시오. 자세한 내용은 다음을 참조하세요. [고급 로깅을 사용하는 방법](#howtoenableadvancedlogging). Adobe Learning Manager 데스크탑 응용 프로그램을 다시 시작하고 오류를 초래하는 단계를 다시 실행하십시오. 도움이 필요하면 최신 응용 프로그램 로그를 Adobe Learning Manager 관리자에게 보내십시오. 자세한 내용은 다음을 참조하세요. [응용 프로그램 로그를 찾는 방법](#howtofindapplicationlogs).

## 응용 프로그램 환경 설정을 재설정하는 방법 {#howtoresetapplicationpreferences}

### Windows {#windows}

1. [실행] 대화 상자를 열려면 **Windows + R** 키.
1. 유형 `**%APPDATA%\\..\\Local\\Adobe\\Learning Manager 1.0**` 그런 다음 Enter 키를 누릅니다.
1. 이름이 지정된 파일 삭제 **preferences.json** 및 **preferences.xml**.

### Mac {#macosx}

1. 파인더를 엽니다.
1. 를 열려면 **다음으로 이동:** 폴더 대화 상자, **Cmd + Shift + G** 키.
1. 유형 `**~/Library/Application Support/Adobe/Learning Manager 1.0**` 그런 다음 Enter 키를 누릅니다.
1. 이름이 지정된 파일 삭제 **preferences.json** 및 **preferences.xml**.

## 응용 프로그램 로그를 찾는 방법? {#howtofindapplicationlogs}

### Windows {#application-logs}

1. 실행 대화 상자를 열려면 키를 누릅니다 **Windows + R** 키.
1. 유형 `**%TEMP%\\elthor**` 그런 다음 Enter 키를 누릅니다.
1. 폴더 정렬 기준: **수정한 날짜** 최신 폴더를 엽니다. 이 폴더에는 최신 응용 프로그램 로그가 포함되어 있습니다.

### Mac {#MacOSX-1}

1. 열기 **파인더**.
1. 를 열려면 **폴더로 이동** 대화 상자에서 키를 누릅니다 **Cmd + Shift + G** 키.
1. 문자 &quot;**/var/folders**&quot;(따옴표 제외)을 입력하고 Enter 키를 누릅니다.
1. &quot;&quot; 검색&#x200B;**엘토르**&quot;검색 창에서 폴더를 엽니다.
1. **수정한 날짜**를 기준으로 폴더를 정렬하고 가장 최근 폴더를 엽니다. 이 폴더에는 최신 응용 프로그램 로그가 포함되어 있습니다.

## Adobe Learning Manager 데스크탑 앱의 이전 설치를 정리하는 방법은 무엇입니까? {#howtocleanuppreviousinstallationsofadobelearningmanagerdesktopapp}

### Windows {#Windows-3}

1. 를 열려면 **대화 상자 실행,**&#x200B;누르기&#x200B;**Windows 키 + R**.
1. regedit and search 를 입력합니다.**HKEY_LOCAL_MACHINE \\SOFTWARE\\Classes\\Installer\\**&quot;(따옴표 제외) 또는 &quot;**HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows\\CurrentVersion\\Installer\\UserData\\S-1-5-18\\Products\\**&quot;(따옴표 제외)을 입력하고 Enter 키를 누릅니다.
1. Adobe Learning Manager라는 폴더를 찾아 이전 설치를 찾습니다. 레지스트리 항목을 삭제합니다.  F3 키를 누르면 키를 찾을 수 있다.

### Mac {#MacOSX-6}

다음 경로에서 파일을 이동합니다.**/Applications/Adobe Learning Manager/Users/Shared/Adobe/Learning Manager Assets/1.0**&quot; 휴지통으로 이동한 다음 휴지통을 비웁니다.
