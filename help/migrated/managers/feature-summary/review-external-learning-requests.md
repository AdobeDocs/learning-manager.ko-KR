# 관리자로서 외부 학습 요청 검토

팀의 학습자가 Adobe Learning Manager에서 외부 학습 요청을 제출하면 플랫폼 내 알림을 받게 됩니다. 제출 세부 정보를 검토하고, 요청을 승인 또는 거부하고, 학습자에 대한 의견을 추가할 수 있습니다.

## 관리자 검토 워크플로우 작동 방식

학습자가 외부 학습 요청을 제출하면 다음과 같은 상황이 발생합니다.

1. 제출을 검토하라는 **앱 내 알림**&#x200B;을 받습니다. 관리자 대시보드의 **외부 학습** 탭에 제출이 표시됩니다.
2. 제출 서류를 열고, 모든 필드와 업로드된 문서를 증거로 검토하고, **승인** 또는 **거부**&#x200B;를 선택합니다.
3. 알림을 받을 때 학습자에게 표시되는 **검토 의견**&#x200B;을 추가할 수 있습니다.
4. 학습자는 귀하의 결정과 함께 **플랫폼 내 알림**&#x200B;을 받습니다.

제출을 승인하면 외부 학습 활동이 **관리자 학습자 성적 증명서**&#x200B;에 추가되고 학습자의 성적 증명서 기록에 나타납니다.

## 제출 서류 검토 및 승인 또는 거부

1. 관리자로 Adobe Learning Manager에 로그인합니다.

2. 왼쪽 탐색 창에서 **외부 학습**&#x200B;을 선택합니다.

3. 제출 목록에서 검토할 요청을 선택합니다. 제출물은 제출된 날짜별로 정렬되며 가장 최근의 항목이 맨 위에 표시됩니다.

4. 전체 제출 검토:

   - 제목, 설명, 날짜, 기간 및 점수

   - 관리자가 구성한 모든 사용자 정의 필드

   - 첨부된 증빙 서류(있는 경우) 확인 또는 다운로드할 첨부 파일 선택

5. **승인** 또는 **거부**&#x200B;를 선택합니다.

6. **주석 검토** 필드에 학습자에 대한 메모를 입력합니다. 이는 선택 사항이지만 요청을 거부할 때 권장되므로 학습자는 수정할 내용을 이해합니다.

7. **제출**&#x200B;을 선택합니다.

학습자는 결정 사항에 대한 앱 내 알림을 받습니다. 제출을 승인하면 이제 학습자 성적 증명서에 나타납니다.

## 전송 대기열 관리

외부 학습 대기열에는 부하직원으로부터 받은 보류 중인 제출 및 과거 제출물이 모두 표시됩니다.

**상태별 필터링**

**상태** 필터를 사용하여 목록 범위를 좁힙니다.

- **모두**- 상태에 관계없이 모든 제출을 표시합니다.

- **검토 대기 중-**&#x200B;은(는) 검토 보류 중인 제출만 표시합니다.

- **승인됨-**&#x200B;은 이미 승인한 제출을 표시합니다.

- **거부됨-**&#x200B;은(는) 거부된 제출을 표시합니다.

**검색 및 정렬**

- **검색** 필드를 사용하여 학습자 이름으로 제출한 항목을 찾습니다.

- 제출물은 기본적으로 제출된 날짜별로 정렬되며 가장 최근의 항목이 맨 위에 표시됩니다.

### 승인 회람 규칙

기본적으로 외부 학습 제출물은 학습자의 직접 관리자에게 전달됩니다. 학습자에게 직접 관리자가 할당되지 않은 경우 다음 규칙이 적용됩니다.

| **학습자에게 관리자가 있습니다** | **학습자가 관리자입니다** | **제출이**(으)로 라우팅됨 |
|---------------------------|-------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| 예 | 아니오 | Direct manager(기본 사례) |
| 예 | 예 | Direct manager(기본 사례) |
| 아니오 | 아니오 | 루트 계정 사용자, 루트 계정 사용자에게 관리자 역할이 있는 경우. 그렇지 않은 경우 제출이 자동으로 승인됩니다. |
| 아니오 | 예 | 루트 계정 사용자, 루트 계정 사용자에게 관리자 역할이 있는 경우. 그렇지 않은 경우 제출은 학습자에게 라우팅됩니다. |

특정 학습자의 관리자 할당에 대한 질문이 있는 경우 계정 관리자에게 문의하십시오.

## 외부 학습 보고 및 성적 증명서 변경

Adobe Learning Manager에서 학습자의 외부 학습 제출이 승인되면 활동이 보고 시스템에 추가되고 책임자 학습자 성적 증명서와 학습자 성적 증명서 모두에 나타납니다.

### 학습자 성적 증명서에 외부 학습이 표시되는 방식

**참고:** 외부 학습을 활성화하면 관리자 학습자 성적 증명서에 **외부 학습 이름**, **완료 설명** 및 각 사용자 지정 필드의 동적 열이 새로 추가됩니다. 사용자 정의 필드 열은 내보내기 끝에 항상 표시됩니다. 학습자 성적 증명서 데이터가 자동화된 보고 또는 BI 도구로 전달되는 경우 추가 열을 처리하도록 해당 파이프라인이 업데이트되었는지 확인합니다.

외부 학습 제출 **승인**&#x200B;만 성적 증명서에 표시됩니다. **승인 대기 중** 또는 **거부됨** 상태의 제출은 성적 증명서 내보내기에 포함되지 않습니다.

책임자 학습자 성적 증명서와 학습자 성적 증명서는 외부 학습 제목을 다르게 처리합니다.

- **관리자 학습자 성적 증명서**&#x200B;에서 외부 학습 제목은 기존 **LP/인증/과정** 열에 배치되며, 열 구조는 다른 학습 활동 유형과 일관되게 유지됩니다.

- **학습자 성적 증명서**(학습자 생성)에서 **모듈** 열 바로 뒤에 **외부 학습 이름**&#x200B;이라는 새 열이 추가됩니다.

책임자가 구성한 사용자 정의 필드는 제출이 승인되면 두 대본 내보내기 모두 끝에서 동적 열로 표시됩니다.

외부 학습용 책임자 학습자 성적 증명서 행의 날짜 기반 필터링은 승인 날짜에 해당하는 **완료 날짜**&#x200B;를 기준으로 합니다.
