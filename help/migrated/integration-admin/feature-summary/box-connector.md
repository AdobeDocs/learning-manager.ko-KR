# Adobe Learning Manager의 Box 커넥터

## 소개

Adobe Learning Manager의 **Box 커넥터**&#x200B;를 사용하면 CSV 파일을 통해 사용자 및 학습 데이터를 자동으로 가져오고 내보내 외부 시스템과 원활하게 통합할 수 있습니다. 외부 시스템은 CSV 파일을 Adobe Learning Manager 관리 Box 계정의 지정된 폴더에 배치할 수 있으며, 여기서 지정된 일정에 따라 자동으로 처리됩니다.

이 커넥터를 사용하여 관리자는 다음을 수행할 수 있습니다.

- CSV 파일에서 내부 사용자 가져오기.
- 사용자 스킬 데이터 및 학습자 성적 증명서를 외부 시스템으로 내보냅니다.
- 지원되는 타사 시스템에서 xAPI 활동 명령문 가져오기.

이 커넥터는 속성 매핑, 예약 동기화 및 온디맨드 실행을 지원하여 조직이 플랫폼 전반에 걸쳐 최신 사용자 및 학습 데이터를 유지 관리할 수 있도록 지원합니다.

## Box 커넥터 구성

Adobe Learning Manager에서 Box 커넥터를 설정하려면:

1. 통합 관리자로 Adobe Learning Manager에 로그인합니다.
2. **Box** 타일 위로 마우스를 가져갑니다.
3. **연결**&#x200B;을 선택합니다.

   ![](assets/box-connector1.png)
   _Box 커넥터를 구성하려면 [연결]을 선택하세요. Box 커넥터를 구성하려면 [연결]을 선택하세요._

4. 조직의 Adobe Learning Manager Box 계정을 관리할 사람의 이메일 주소를 입력합니다.
5. **연결**&#x200B;을 선택합니다.



### 계정 활성화

1. Adobe Learning Manager은 제공된 이메일 ID로 암호 재설정 링크를 전송합니다.
2. 사용자는 Box 계정에 처음 액세스하기 전에 암호를 재설정해야 합니다.

>[!NOTE]
>
>Adobe Learning Manager 계정당 하나의 Box 계정만 구성할 수 있습니다.

**개요** 페이지에서 다음 작업 중 하나를 선택합니다.

- **내부 사용 가져오기**
- **xAPI 활동 보고서 가져오기**
- **사용자 스킬 내보내기**
- **학습자 성적 증명서 내보내기**
- **xAPI 활동 보고서 내보내기**

연결되면 Box 커넥터가 Adobe Learning Manager과 외부 시스템 간에 데이터를 동기화할 준비가 됩니다.

## 내부 사용자 가져오기

사용자 가져오기 기능을 사용하면 HR 시스템 및 기타 외부 소스에서 Adobe Learning Manager으로 직원 데이터를 자동으로 동기화할 수 있습니다.

### 맵 속성

속성 매핑은 외부 데이터와 Adobe Learning Manager의 지원되는 데이터 구조 간에 연결을 만들어 데이터가 올바른 필드에 저장되도록 합니다. 이 과정은 꼭 필요합니다.

속성을 매핑하려면 다음을 수행합니다.

1. Box 커넥터 페이지에서 **내부 사용자**&#x200B;를 선택합니다.
2. **열 매핑**&#x200B;을 선택합니다.
3. **특성 매핑** 페이지에서 다음을 수행합니다.
   - 왼쪽에는 Adobe Learning Manager의 필수 필드가 표시됩니다.
   - 오른쪽에는 CSV 열 이름이 표시됩니다. 처음에는 이 쪽에 빈 드롭다운이 포함되어 있습니다.
   - 샘플 CSV 파일을 업로드하려면 **CSV 선택**&#x200B;을 선택합니다. 그러면 오른쪽 드롭다운이 CSV의 열 이름으로 채워집니다. 샘플 CSV를 가져오려면 [이 문서](https://experienceleague.adobe.com/ko/docs/learning-manager/using/integration/migration-manual#csv)를 참조하세요.
   - 각 Adobe Learning Manager 필드를 해당 CSV 열로 매핑합니다.

   ![](assets/box-connector2.png)
   _왼쪽의 Adobe Learning Manager 필드와 오른쪽의 CSV 열 드롭다운을 표시하는 특성 매핑 인터페이스_

4. **저장**&#x200B;을 선택하여 매핑을 완료합니다.

저장 후 구성된 계정은 책임자 앱에서 데이터 소스로 나타납니다. 그러면 관리자는 가져오기를 예약하거나 수동 동기화를 트리거할 수 있습니다.

### xAPI 명령문 가져오기

xAPI 명령문 가져오기를 사용하면 외부 학습 데이터를 Adobe Learning Manager으로 가져와 자세한 학습 활동 추적이 가능합니다.

_소스 구성_

xAPI 소스 구성은 외부 학습 시스템과 Adobe Learning Manager의 활동 추적 간의 연결을 설정합니다.

소스를 구성하려면 다음을 수행합니다.

1. Xapi 구성 섹션으로 이동합니다.
2. 구성 목록에서 **새 구성 추가**&#x200B;를 선택합니다.
3. **이름** 및 **원본 파일 이름**&#x200B;을 입력하십시오.
   - 이름: 이 xAPI 소스에 대한 설명 식별자(예: LMS 통합 또는 외부 교육 시스템)
   - 소스 파일 이름: Box 폴더에 업로드될 정확한 파일 이름입니다(파일 확장자를 포함하여 정확히 일치해야 함).

   ![](assets/box-connector3.png)
   _이름 필드 및 원본 파일 이름 필드를 표시하는 구성 양식_

4. **저장**&#x200B;을 선택하여 기본 구성을 만듭니다.

_필터 추가(선택 사항)_

필터를 사용하면 특정 기준에 따라 xAPI 명령문을 선택적으로 가져올 수 있습니다.

소스에 필터를 추가하려면 다음을 수행하십시오.

1. 왼쪽 창에서 **필터**&#x200B;를 선택합니다.
2. **새 필터 추가**&#x200B;를 선택합니다.
3. 다음을 구성합니다.
   - **이름:** 필터 규칙에 대한 설명 이름입니다.
   - **조건:** 비교 연산자(같음, 포함, 보다 큼 등)

   ![](assets/box-connector4.png)
   _이름 및 조건 필드를 표시하는 필터 만들기 대화 상자_

4. **새 필터 추가**&#x200B;를 선택하여 필터를 더 추가합니다.
5. **작업** 열에서 필요에 따라 **저장** 또는 **삭제**&#x200B;를 선택합니다.
6. 필터를 추가한 후 **저장**&#x200B;을 선택합니다.

## 가져오기 예약

자동 스케줄링은 수작업 없이 일관된 데이터 동기화를 보장하여 현재 학습 활동 기록을 유지합니다.

가져오기를 예약하려면 다음을 수행하십시오.

1. 왼쪽 창에서 **일정 구성**&#x200B;을 선택합니다.

   ![](assets/box-connector5.png)
   _사용 옵션 및 타이밍 컨트롤을 표시하는 구성 예약 페이지_

2. **이 연결을 사용하여 xAPI 문 가져오기 사용**&#x200B;을 선택합니다.
3. 자동 가져오기를 설정하려면 **예약 사용**&#x200B;을 선택하세요.
4. 다음 예약 매개 변수를 설정합니다.

   - 예약된 가져오기를 시작해야 하는 **시작 날짜:**.
   - **시간:** 가져오기 실행 시간입니다.
   - **다음 이후 반복:** 가져오기를 실행할 빈도(매일, 매주, 사용자 지정 간격)입니다.
5. **저장**&#x200B;을 선택합니다.

## 요청 시 실행(선택 사항)

온디맨드 실행은 정규 스케줄링된 작업 이외의 데이터를 즉시 임포트합니다.

온디맨드 가져오기를 사용하는 경우:

- 스케줄링 전에 새 구성 테스트
- 긴급 또는 시간에 민감한 데이터 업데이트 처리.
- 일회성 데이터 마이그레이션 또는 수정 처리
- 가져오기 문제 해결

xAPI 명령문을 수동으로 가져오려면 다음을 수행합니다.

1. 왼쪽 창에서 **온디맨드**&#x200B;를 선택합니다.
2. **실행**&#x200B;을 선택합니다.

## 실행 상태 보기

상태 모니터링을 통해 가져오기 작업의 사전 예방적 관리와 문제의 빠른 파악이 가능합니다.

실행 상태를 보려면 다음을 수행합니다.

1. 모든 가져오기 실행 목록을 보려면 **실행 상태**&#x200B;를 선택하세요.
2. 상태 페이지에는 다음이 표시됩니다.

   - 가져오기 작업이 시작된 **시작 날짜:**
   - **기간:** 처리에 필요한 총 시간
   - **가져오기 유형:** 가져오기가 예약되었는지 아니면 온디맨드로 예약되었는지 여부
   - **현재 상태:** 실시간 상태 정보
      - **진행 중:** 가져오기가 현재 실행 중입니다.
      - **완료됨:** 레코드 수가 포함된 완료 성공
      - **실패:** 진단 정보에 오류가 발생했습니다.
