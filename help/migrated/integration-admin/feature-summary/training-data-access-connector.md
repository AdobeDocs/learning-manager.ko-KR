# Adobe Learning Manager의 교육 데이터 액세스 커넥터

## 소개

**교육 데이터 액세스 커넥터**&#x200B;를 사용하면 독립형 또는 **Adobe Experience Manager(AEM) 사이트**&#x200B;로 빌드된 사용자 지정 인터페이스에 통합될 수 있는 헤드리스 학습 환경을 만들 수 있습니다. 이 커넥터는 검색 및 필터 기능과 함께 학습자에게 최신 교육 콘텐츠를 검색하고 표시하는 데 유용합니다.

>[!IMPORTANT]
>
>- 이 기능은 Adobe Learning Manager이 Adobe Experience Manager에 **추가 기능**(으)로 판매되는 경우에만 사용할 수 있습니다.
>- 이 커넥터를 통해 검색된 과정 데이터는 24시간마다 새로 고쳐집니다
>- 이 커넥터는 헤드리스 또는 AEM 기반 임시 방문 경험을 구축하기 위한 셀프 서비스가 아닙니다. 사용 사례에 맞는 접근 방법을 계획하려면 Adobe에 문의하십시오.

## 작동 방식

커넥터가 활성화되면 Adobe Learning Manager은 과정, 학습 경로 및 인증서와 같은 교육 메타데이터를 제공하는 공개 API 세트를 표시합니다. 이러한 API를 사용하여 교육 콘텐츠를 표시하고 검색 및 필터링 기능을 지원하는 사용자 정의 브랜디드 프런트엔드를 구축할 수 있습니다.

## 교육 데이터 액세스 커넥터 구성

Adobe Learning Manager을 데이터 스토리지 및 검색 시스템과 통합하여 AEM Sites 또는 기타 헤드리스 환경에 교육 메타데이터를 푸시할 수 있습니다.

커넥터를 구성하려면:

1. 통합 관리자로 Adobe Learning Manager에 로그인합니다.
2. **교육 데이터 액세스** 타일 위로 마우스를 가져간 다음 **연결**&#x200B;을 선택합니다.

   ![](assets/training-data-access-connector1.png)
   _연결을 선택하여 교육 데이터 액세스 커넥터를 구성하세요_

3. **연결 이름**&#x200B;을 입력하십시오.
4. **인터페이스 유형** 선택:

   - **기본 Learning Manager**: 기본적으로 제공되는 표준 로그인 경험입니다.
   - **헤드리스 인터페이스**: 로그인하지 않은 헤드리스 프런트 엔드에 공개 API를 표시하는 프리미엄 옵션입니다.

   ![](assets/training-data-access-connector2.png)
   _교육 데이터 액세스 커넥터 구성에 필요한 세부 정보 입력_

5. **연결**&#x200B;을 선택합니다. Adobe Learning Manager은 **기본 URL** 및 **CDN URL**&#x200B;을(를) 자동으로 생성합니다. 맞춤형 사이트 또는 앱에서 이러한 URL을 사용하여 교육 데이터를 가져옵니다.

>[!NOTE]
>
>프리미엄 플랜의 고객은 표준 고객과 다른 API URL을 받습니다.

## 교육 메타데이터 내보내기

교육 메타데이터를 내보내려면 다음을 수행합니다.

1. 커넥터 페이지에서 **교육 메타데이터 내보내기**&#x200B;를 선택합니다.
2. **이 연결을 사용하여 교육 메타데이터 내보내기 사용**&#x200B;을 선택하여 교육 데이터를 검색 및 검색 시스템에 푸시합니다.
3. **예약 사용**&#x200B;을 선택하고 시작 날짜, 시간 및 간격을 설정합니다.

   ![](assets/training-data-access-connector3.png)
   _교육 메타데이터 내보내기 예약_

4. **저장**&#x200B;을 선택합니다.

   - 모든 과정, 학습 경로 및 인증서 이미지를 **CDN**&#x200B;에 자동으로 업로드합니다.
   - 또한 연결된 메타데이터를 검색 시스템으로 내보냅니다.

### 온디맨드 내보내기

- **온디맨드 내보내기 실행:** **온디맨드**(으)로 이동하고 **시작 날짜**&#x200B;를 설정한 다음 **실행**&#x200B;을 선택하여 필요할 때 내보내기를 실행합니다.
- **실행 상태 확인:** **실행 상태** 페이지에서 내보내기 진행률과 기록을 봅니다.

## AEM에서 웹 사이트 빌드 및 게시

헤드리스 또는 AEM Sites 기반 웹 사이트에 교육 데이터를 표시하려면:

1. [Adobe의 GitHub 저장소](https://github.com/adobe/adobe-learning-manager-reference-site/releases/tag/1.0.0)&#x200B;(필수 구성 요소)에서 **AEM 패키지를 설치**&#x200B;합니다.
2. **기본 URL**, **CDN URL**, **클라이언트 ID**, **클라이언트 암호** 및 **관리자 새로 고침 토큰**&#x200B;을 사용하여 AEM에서 구성을 만드십시오.
3. AEM 구성 요소를 사용하여 사이트를 빌드합니다.
4. 학습자용 Publish 사이트
5. 전체 설정에 대한 자세한 내용은 [이 문서](https://experienceleague.adobe.com/en/docs/learning-manager/using/integration/aem-sites/adobe-learning-manager-integration-aem) 및 [이 문서](https://experienceleague.adobe.com/en/docs/learning-manager/using/integration/aem-sites/integrate-aem-learning-manager)를 참조하세요.

### 학습자 경험

사이트가 활성 상태가 되면 다음을 수행합니다.

- 웹 사이트에는 검색 시스템을 통해 Adobe Learning Manager에서 검색된 모든 **과정**, **학습 경로** 및 **인증서**&#x200B;가 표시됩니다.
- **로그인하지 않은** 학습자는 과정 세부 정보를 검색하고 볼 수 있습니다.
- 학습자가 강의, 학습 경로 또는 인증서를 등록하기 위해 클릭하면 등록을 완료하고 교육을 시작하려면 **로그인**&#x200B;하라는 메시지가 표시됩니다.

## 임시 방문 모드

임시 방문 모드에서는 로그인하지 않은 사용자를 위한 실시간 경험을 만들 수 있습니다. 예를 들어 임시 방문 경험은 마케팅 캠페인의 시작 페이지 역할을 하여 등록을 독려합니다.

**교육 데이터 액세스** 커넥터를 사용하여 Adobe Learning Manager의 임시 방문 환경을 구성할 수 있습니다. 커넥터는 다음 기능을 제공합니다.

- 표준 제품
- 프리미엄 제품

### 표준 제품

표준 제품은 Adobe Learning Manager의 기본 버전을 빌드하는 것입니다. 사용자는 데모 전용, 로그인하지 않은 헤드리스 경험을 구축할 수 있습니다. 데모 헤드리스 경험은 확장할 수 없으므로 프로덕션 환경에서 사용해서는 안 됩니다.

### 프리미엄 제품

프리미엄 제품은 사용자가 **교육 데이터 액세스** 커넥터로 구성된 헤드리스 인터페이스를 빌드하는 데 도움이 됩니다. 이를 통해 사용자는 이름, 설명, 작성자, 스킬, 기간 등과 같은 강의 및 학습 경로 세부 정보에 대한 실시간 데이터를 가져올 수 있습니다. 혼합 학습 시나리오의 경우 실시간 인원 제한, 사용 인원, 대기자 명단 제한 및 대기자 명단 수도 표시됩니다. 고객은 이러한 API를 사용하여 검색 및 필터링 기능과 로그인하지 않은 학습자를 위한 전체 강의 요약을 만들 수 있습니다.

고객은 프리미엄 플랜을 구매하여 고도로 확장 가능한 로그인 경험을 구축할 수 있습니다.

>[!NOTE]
>
>지원 팀 또는 CSM에 문의하여 프리미엄 플랜을 구매하십시오.

사용자가 플랜을 구매한 후 CSM 팀에서 프리미엄 플랜을 활성화합니다. 사용자는 교육 데이터 액세스 커넥터를 사용하여 앞에서 언급한 기능으로 로그인하지 않는 경험을 설정할 수 있습니다.
