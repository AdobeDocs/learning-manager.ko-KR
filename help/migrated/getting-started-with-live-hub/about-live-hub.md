# Live Hub 정보

Live Hub는 가상 교실을 Adobe Learning Manager에 기본적으로 제공하므로 LMS(교육 관리 시스템)와 회의 도구 간에 전환이 없습니다. 모든 세션은 자동으로 추적되며, 풍부한 세션 대시보드는 강사에게 출석을 넘어 학습자의 참여 방식을 보여줍니다.

라이브 교육을 위해 의도적으로 빌드된 라이브 허브의 AI 상담사는 내내 강사와 함께 작업합니다. 설문조사를 제안하고, 참여자 질문과 답변을 찾아내고, 소회의실을 실시간으로 모니터링하고, 세션 요약을 생성하여 교수자가 교육 전달에 집중할 수 있도록 한다. 그리고 학습자를 위해 레코딩이 자동으로 탐색 가능한 주제로 분할되어 라이브에 참여하는 것처럼 구조화된 유용한 리플레이가 됩니다.

Live Hub는 강사가 매력적인 가상 교육을 제공하고 조직이 대규모 대화형 라이브 교육을 실행하는 데 도움이 됩니다. 곧 Adobe Learning Manager 8월 릴리스에서 라이브 허브(베타)가 출시될 예정입니다.
