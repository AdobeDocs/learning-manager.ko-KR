# Live Hub 정보

Live Hub는 가상 교실을 Adobe Learning Manager에 기본적으로 제공하므로 LMS(교육 관리 시스템)와 회의 도구 간에 전환이 없습니다. 모든 세션은 자동으로 추적되며, 풍부한 세션 대시보드는 강사에게 출석을 넘어 학습자의 참여 방식을 보여줍니다.

라이브 교육을 위해 의도적으로 빌드된 라이브 허브의 AI 상담사는 내내 강사와 함께 작업합니다. 그들은 강사가 교육 세션을 전달하는 데 집중할 수 있도록 설문 조사를 제안하고, 참가자 질문을 감지하고 답변을 제안하며, 소회의실을 실시간으로 모니터링하고 세션 요약을 생성합니다. 학습자에게 레코딩은 탐색 가능한 주제로 자동으로 분할되어 라이브에 참여하는 것처럼 구조화된 유용한 리플레이가 됩니다.

라이브 허브는 강사가 매력적인 가상 교육을 제공하고 조직이 규모에 맞게 대화형 라이브 교육을 실행하는 데 도움을 줍니다.

>[!NOTE]
>
>Live Hub는 Adobe Learning Manager의 향후 **8월 릴리스**&#x200B;에서 사용할 수 있습니다. 기능이 릴리스되면 라이브 허브에 대한 설명서가 제공됩니다.
