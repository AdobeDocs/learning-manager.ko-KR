# 작업 지원

**작업 지원**&#x200B;은(는) 등록이나 완료 조건 없이 학습자가 액세스할 수 있는 교육 콘텐츠 저장소입니다. 학습자는 이러한 작업 지원을 참조하여 조직 내 활동이나 작업 수행을 지원받을 수 있습니다.

작업 지원을 단독으로 받거나 Learning Manager 강의 수료와 함께 받을 수 있습니다.

작성자는 학습자용 작업 지원을 생성할 수 있습니다. 학습자가 작업을 완료하기 위해 계속 사용할 수 있는 팁, 체크리스트, 가이드와 같은 참조 자료를 작업 지원을 사용하여 학습자에게 제공합니다.

## 작업 지원 생성 {#createjobaid}

1. 작성자 로그인에서 왼쪽 창의 **[!UICONTROL 작업 지원]**&#x200B;을 선택합니다.
1. 표시되는 페이지의 오른쪽 상단 모서리에 있는 **[!UICONTROL 만들기]**&#x200B;를 선택합니다.
1. 이름, 설명 및 태그를 입력합니다. 스킬 및 관련 레벨을 선택합니다. 각 강의에 할당하는데 다른 작성자가 작업 지원에 접근하지 않게 하려면 콘텐츠를 비공개로 선택합니다.

   기존 스킬만 작업 지원에 사용할 수 있습니다. 스킬은 필수 사항이 아닙니다.

1. 콘텐츠 섹션에 작업 지원 콘텐츠를 업로드할 수 있습니다.

   업로드에 비디오, pdf, pptx 및 docx 파일 형식이 지원됩니다. 프로젝트 zip 파일 또는 대화형 콘텐츠는 업로드할 수 없습니다.

1. 작업 지원에 대한 지속 기간을 분 단위로 입력하십시오.
1. **[!UICONTROL 저장]**&#x200B;을 클릭합니다.

   작업 지원이 게시됩니다.

## 다른 언어로 작업 지원 추가 {#addcontentfordifferentlanguages}

1. 다른 언어로 작업 지원을 추가하려면 **새 언어 추가** 탭을 선택하고 필요한 언어를 선택합니다. 이 접근법을 사용하면 콘텐츠에 대한 다중 언어 지원을 추가할 수 있습니다.

   ![](assets/add-new-languagetab.png)

   *콘텐츠에 새 언어 추가*

1. 새 언어에 대해 작업 지원 업로드 프로세스를 반복합니다.
1. 언어를 제거하려면 **[!UICONTROL 새 언어 추가]** 탭을 선택하고 선택을 취소합니다.

   변경한 후 저장 을 선택합니다.

## 지원되는 작업 지원 유형 {#typesofsupportedjobaids}

작업 지원에 지원되는 파일 형식은 다음과 같습니다.

* PDF
* PPT
* PPTX
* XLS
* XLSX
* DOC
* DOCX
* 모든 동영상 파일 형식

>[!NOTE]
>
>Zip 파일 및 이미지 파일은 지원되지 않습니다.

## 다국어 작업 지원

Adobe Learning Manager(ALM)의 다국어 작업 지원을 통해 작성자와 관리자는 단일 작업 지원 항목 내에서 지원 문서, 안내선 또는 리소스를 다국어로 제공할 수 있습니다. 다른 지역의 학습자가 선호하는 언어로 관련 자료에 액세스할 수 있어 이해력, 준수성 및 사용자 경험이 향상됩니다.

**사용 사례**

* 글로벌 인력 지원: 다양한 인력을 대상으로 안전 매뉴얼, 프로세스 가이드 또는 참조 문서를 다국어로 제공합니다.
* 규정 준수: 모든 직원이 동일한 규정 준수 문서를 모국어로 받도록 합니다.
* 일관된 온보딩: 전 세계 신규 채용에 대해 온보딩 체크리스트 또는 FAQ를 현지 언어로 제공합니다.
* 중복 감소: 작업 지원의 모든 언어 버전을 단일 항목으로 관리하여 업데이트 및 보고를 단순화합니다.

### 주요 기능

* 다중 언어 지원: 단일 작업 지원 내에 지원되는 각 언어에 대한 고유 파일 또는 URL을 첨부합니다.
* 한글화된 이름 및 설명: 작업 지원의 이름 및 설명을 각 언어로 입력합니다.
* 통합 관리: 모든 언어 버전을 한 곳에서 편집, 업데이트 및 보고합니다.
* 이전 버전과의 호환성: 기존 단일 언어 작업 지원은 새 파일이 업로드될 때까지 추가된 모든 언어에 자동으로 복제됩니다.

### 다국어 작업 지원 만들기

1. 작성자 역할로 이동하고 작업 지원을 선택합니다.
2. 작업 지원 생성을 선택합니다.
3. 작업 지원 이름 및 설명을 기본 언어로 입력합니다.
4. 기본 언어에 대한 주요 컨텐트 파일 또는 URL을 추가합니다.
5. 작업 지원을 저장합니다.

### 추가 언어 추가

1. 작업 지원 편집기에서 [언어 추가]를 선택합니다.
2. 목록에서 원하는 언어를 선택합니다.
3. 추가된 각 언어에 대해:
   * 현지화된 이름과 설명을 입력합니다.
   * 해당 콘텐츠 파일을 업로드하거나 언어별 URL을 제공합니다.
4. 필요한 모든 언어에 대해 반복합니다.

### 언어 편집 및 관리

1. 특정 언어에 대한 파일이나 설명을 업데이트하려면 언어 탭을 선택하고 필요에 따라 변경합니다.
2. 작업 지원이 게시된 후 언어가 추가되면 고유한 파일이 업로드될 때까지 원본 파일이 자동으로 새 언어에 할당됩니다.
3. 필요에 따라 모든 언어의 파일을 제거하거나 대체합니다.

### Publish 및 학습자 경험

1. 모든 언어와 파일을 추가한 후 작업 지원을 게시합니다.
2. 학습자는 해당 파일 또는 URL과 함께 작업 지원을 선택한 콘텐츠 언어로 볼 수 있습니다.
3. 학습자의 언어를 사용할 수 없는 경우 기본 언어 파일이 표시됩니다.

### 보고

* 작업 지원 보고서를 다운로드하여 각 작업 지원과 관련된 모든 파일 및 언어의 세부 정보를 볼 수 있습니다.
* 보고서에는 추적을 위한 언어, 파일 이름 및 사용 데이터가 포함됩니다.

### 모범 사례

* 이름, 설명 및 콘텐츠 파일에 대한 정확한 번역을 제공합니다.
* 여러 언어에서 일관성을 유지하기 위해 파일을 정기적으로 검토하고 업데이트합니다.
* 명확한 이름 지정 규칙을 사용하여 다른 언어의 파일을 구분합니다.
* 콘텐츠 언어를 전환하여 올바른 파일 전달 여부를 확인하여 학습자 경험을 테스트합니다.

다국어 작업 지원을 사용하면 단일 항목으로 글로벌 고객에게 지원 리소스를 제공하고, 중복을 줄이며, 모든 학습자가 선호하는 언어로 올바른 정보를 받을 수 있도록 할 수 있습니다. 이 기능은 Adobe Learning Manager의 접근성, 규정 준수 및 관리 효율성을 향상시킵니다.

## 작업 지원 철회/재승인 {#withdrawrepublishjobaids}

작업 지원 옆에 있는 &#39;설정&#39; 아이콘을 클릭하고 &#39;철회&#39;를 선택하여 작업 지원 승인을 철회할 수 있습니다.

![](assets/job-aid-withdraw.png)

*게시된 작업 지원 편집, 철회 또는 미리 보기*

철회 탭을 클릭하면 철회된 작업 지원이 보입니다. 설정 아이콘을 클릭하고 승인을 선택하면 철회된 작업 지원을 재승인할 수 있습니다.

## 작업 지원에서 HTML 패키지 지원

작업 지원에서는 이제 새로운 유형의 콘텐츠로 표준 HTML 패키지를 지원합니다. 이러한 향상된 성능을 통해 학습자가 작업 지원 플레이어에서 HTML 패키지 보기를 열거나 다운로드할 수 있습니다.

작업 지원을 만들 때 작성자는 지원되는 다른 파일 형식과 함께 표준 HTML 패키지를 업로드할 수 있습니다.

![](assets/html-job-aid.png)

*HTML 패키지 지원*

HTML 패키지에는 다음 사항이 반드시 포함되어야 합니다.

* Index.html 파일.
* Index.html 파일은 zip 파일의 루트 폴더에 있어야 합니다.

Index.html 파일의 위치에서 zip 파일로 업로드할 콘텐츠를 지정합니다.

모든 콘텐츠, 리소스 및 자산은 HTML 패키지에서 참조되어야 하며 Index.html을 통해 액세스할 수 있어야 합니다.

작업 지원을 생성한 후 강의를 생성하는 동안 작업 지원을 강의에 추가할 수 있습니다.
