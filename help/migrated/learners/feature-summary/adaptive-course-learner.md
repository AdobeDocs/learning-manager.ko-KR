# 학습자를 위한 적응형 강의

## 소개

Adobe Learning Manager의 적응형 과정은 작성자가 학습 요구 사항을 파악하고 학습 요구 사항에 따라 사용자 정의해야 하는 과정으로, 책임자가 이 기능을 활성화하면 됩니다. 자세한 내용은 [적응 과정](/help/migrated/administrators/feature-summary/adaptive-course-admin.md)을 참조하세요.

## 학습자로 적응형 강의 수강

Adobe Learning Manager에서 적응형 강의를 수강하면 귀하가 속한 사용자 그룹을 기반으로 개인화된 모듈 세트가 표시됩니다.

### 적응형 강의를 열면 표시되는 항목

적응형 강의를 열면 관련된 모듈만 표시됩니다. 다른 사용자 그룹에 할당된 모듈은 전혀 표시되지 않습니다.

완료 진행률 표시줄 및 &quot;X of Y 모듈 완료됨&quot; 수는 필요한 모듈만 반영합니다. 동일한 강의에 등록된 동료는 사용자 그룹을 기반으로 모듈 개수가 다르고, 필수 및 선택 사항의 조합이 다르거나, 완전히 다른 콘텐츠 세트를 볼 수 있습니다.

### 적응 과정에 등록

적응 과정에 등록하는 것은 모든 과정에 등록하는 것과 동일한 방식으로 작동합니다. 직접 카탈로그에서 등록할 수도 있고 관리자가 등록할 수도 있습니다.

![](assets/image_0010.png)

현재 사용자 그룹을 기반으로 강의에 표시되는 모듈이 없는 경우에는 등록할 수 없습니다. 이러한 경우 관리자에게 문의하십시오. 이는 사용자 그룹 할당을 업데이트해야 함을 의미할 수 있습니다.

### 플레이어에서 모듈 탐색

강좌 플레이어를 열면 왼쪽의 목차에는 볼 수 있는 모듈만 표시됩니다. 내비게이션 버튼을 사용하여 순서대로(순서가 지정된 강의의 경우) 모듈을 이동하거나 순서에 관계없이(순서가 지정되지 않은 강의의 경우) 모듈에 액세스할 수 있습니다.

다른 학습자의 사용자 그룹에 속하는 모듈은 목차에 표시되지 않습니다.

### 완료 요구 사항 이해

**필수** 레이블이 지정된 모든 모듈을 완료하면 과정을 완료합니다. 옵션 모듈은 완료를 차단하지 않습니다. 완료 단계에 영향을 주지 않고 완료 과정을 완료하거나, 다시 방문하거나, 완전히 건너뛸 수 있습니다.

### 강의실 또는 가상 강의실 세션이 가득 찼을 경우 어떻게 됩니까?

강의실 또는 가상 강의실 모듈이 표시되지만 남은 좌석이 없는 경우, 강의에 등록되고 해당 특정 세션의 대기자 명단에 등록됩니다. 계속해서 다른 모든 모듈에 액세스하고 즉시 완료할 수 있습니다. 전체 세션만 보류 중입니다.

세션에 대기자 명단에 등록되는 동안:

* 강좌 플레이어에 세션이 표시되지만 시작하거나 액세스할 수 없습니다.
* 전체 강의 진행은 다른 모듈을 통해 정상적으로 계속됩니다.
* 좌석을 사용할 수 있게 되고 대기자 명단에서 벗어나면 알림을 받게 됩니다.

시트가 할당되면 세션에 액세스할 수 있게 되고, 사용자는 세션에 참여하거나 완료할 수 있습니다.

### 과정 중간에 프로필이 변경되면 어떻게 됩니까?

적응형 강의 환경은 사용자 그룹과 연결됩니다. 역할이나 위치가 변경되는 경우와 같이 등록 중에 사용자 그룹이 변경되면 모듈이 자동으로 업데이트됩니다.

* 업데이트된 사용자 그룹에서 이전에 숨겨진 모듈이 표시되는 경우 해당 모듈이 강의에 표시됩니다. 새 프로필에 필요한 경우 해당 항목이 완료 요구 사항에 추가됩니다.
* 업데이트된 사용자 그룹에 더 이상 모듈이 포함되지 않으면 보기에서 제거됩니다. 해당 모듈이 이전에 필요했지만 더 이상 필요하지 않은 경우 완료 횟수에서 제거됩니다.
* 이미 완료한 모든 모듈은 사용자 그룹이 변경되더라도 완료로 표시됩니다. 새로 표시되는 모듈이 이전에 완료되지 않은 경우 완료된 작업을 다시 실행할 필요가 없습니다.
* 완료된 모듈의 시작 날짜, 완료 날짜 및 전체 진행 기록은 프로필 변경 시 그대로 유지됩니다.

### 과정을 완료한 후에는 어떻게 됩니까?

필요한 모든 모듈을 완료하면 강의가 완료로 표시됩니다. 해당 시점에 해당되는 배지 또는 게임화 점수를 받게 됩니다.

완료 후 사용자 그룹이 변경되고 새로운 필수 모듈을 사용할 수 있게 되면 완료 내용이 롤백될 수 있으며 추가 모듈을 완료해야 할 수 있습니다.

이러한 경우:

* 원작의 완성은 대본에 기록 기록으로 남아 있습니다.
* 새 필수 작업만 표시된 진행 중 상태로 강의가 다시 열립니다.
* 새로 필요한 모듈만 완료하면 됩니다.
* 완료 새로 고침이 트리거되면 알림을 받습니다.
