# Adobe Learning Manager 원클릭 등록

전자 메일, 채팅 또는 다른 메시징 플랫폼을 통해 관리자로부터 딥 링크를 받아 교육 모듈 또는 강의에 등록합니다. 링크를 선택하여 추가 단계 없이 즉시 등록하고 모듈을 실행합니다.

## 학습자를 등록하고 학습자를 강의로 리디렉션

다음 시나리오 및 해당 작업에 주의합니다. 학습자가 받은 딥 링크를 클릭할 경우, 진행 중인 시나리오에 따라 다음 중 하나가 발생합니다.

| SN | 시나리오 | 액션 |
| --- | --- | --- |
| 1 | 단일 모듈 과정 및 비순차 | 자동 등록되고 플레이어가 열립니다. 비디오를 재생하려면 [재생]을 눌러야 합니다. |
| 2 | 다중 모듈 과정 및 비순차 | 자동으로 등록되고 딥 링크가 연결된 특정 모듈에 연결됩니다. 비디오를 재생하려면 [재생]을 눌러야 합니다. |
| 3 | 다중 모듈 과정 및 순차 | 앞의 과정을 완료하지 않은 경우 현재 과정을 시작하기 전에 반드시 앞의 과정을 완료해야 한다는 메시지가 포함된 개요 페이지가 표시됩니다. 이 경우 자동 등록되고 과정의 첫 번째 모듈에 연결됩니다. |
| 4 | 강의에 전제 조건이 있음 | 필수 구성 요소가 완료되면 Player에서 모듈을 시작할 수 있습니다. [재생]을 눌러 비디오를 재생합니다. 필수 강의를 완료하지 않은 경우, 다음 메시지가 표시됩니다. &quot;필수 강의를 건너뛸 수 없습니다. 이 과정을 시작하기 전에 모든 필수 과정을 완료하십시오. |
| 5 | 대기자 명단 포함 과정 | 등록됩니다. 강의 개요 페이지가 나타나고 대기자 명단에 추가됩니다. 강의 대기자 명단에 등록되었다는 메시지가 표시됩니다. |
| 6 | 딥 링크가 생성된 강의에 이미 등록되어 있는 경우. | 모듈에 접속하면 플레이어가 시작됩니다. [재생]을 눌러 비디오를 재생합니다. |

## 인증 상태 및 딥 링크 비헤이비어

위의 시나리오 외에도 인증(로그인 및 로그아웃) 상태와 관련된 두 개의 사용자 동작이 더 있습니다.

* 딥 링크를 클릭하고 Adobe Learning Manager에 로그인하지 않은 경우에는 로그인 페이지로 리디렉션됩니다. 로그인하면 해당 딥 링크가 사용자를 특정 모듈로 리디렉션합니다.

* 딥 링크를 클릭하고 Adobe Learning Manager에 로그인하면 모듈로 직접 리디렉션됩니다.

## 모바일 및 웹에 대한 몰입형 지원

다음 시나리오는 모바일 및 웹 버전의 Adobe Learning Manager 모두에 적용됩니다.

* 데스크탑 버전의 Adobe Learning Manager이 없고 딥 링크를 받은 경우, 해당 링크를 선택하면 웹 또는 모바일 버전에 로그인하지 않은 경우 브라우저에서 Adobe Learning Manager의 로그인 페이지로 리디렉션됩니다.
* Adobe Learning Manager의 데스크탑 버전이 없는 경우 딥 링크를 받는다면 이미 웹 또는 모바일 버전에 로그인한 경우에는 링크를 선택하면 브라우저에서 Adobe Learning Manager 모듈로 바로 리디렉션됩니다.

## 브라우저 제한 자동 재생

딥 링크를 클릭하면 로그인하지 않은 경우 모듈 또는 로그인 페이지로 리디렉션됩니다. 모듈이 비디오인 경우 플레이어는 비디오를 자동으로 재생하지 않습니다. 모든 경우에 브라우저는 플레이어가 비디오를 자동으로 재생하지 못하도록 합니다. 비디오를 재생하려면 [수동 재생]을 눌러야 합니다.
