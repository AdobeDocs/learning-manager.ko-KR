# 소개

Virtual Coach는 이론적 지식과 실제 응용 프로그램 사이의 중요한 간극을 좁히기 위해 설계된 Adobe Learning Manager 내의 혁신적인 기능입니다. 수동적인 학습방법, 동영상, 퀴즈, 읽기 자료에만 의존하기보다는 학습에 대한 만족도를 높일 수 있다. Virtual Coach는 학습자에게 이해도가 높은 대화를 연습하고 소프트 스킬을 학습할 수 있는 대화형 몰입형 환경을 제공합니다.

Virtual Coach를 &quot;24시간 연중무휴 실습 파트너&quot; 및 &quot;커뮤니케이션을 위한 비행 시뮬레이터&quot;로 생각해 보십시오. 조종사가 실제 항공기를 다루기 전에 비행 시뮬레이터에서 수천 시간을 기록하는 것처럼, 직원은 실제 고객, 고객 또는 팀원과 유사한 상황을 마주치기 전에 AI 기반 인물과의 복잡한 대화를 연습할 수 있다. Virtual Coach는 학습자의 입력에 실시간으로 조정하여 사용자가 말한 내용(지식)과 이를 말한 방법(전달 및 스타일)에 대해 데이터 기반의 즉각적인 피드백을 제공합니다.
